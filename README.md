# 基于Vue的todo-list应用

## 项目初始化

> 推荐使用`cnpm` ，速度会非常快

```
npm install -g cnpm --registry=https://registry.npm.taobao.org
```

```
cnpm install
```

### 编译并热加载以进行开发
```
cnpm run serve
```

### 构建生产
```
cnpm run build
```

### 运行测试
```
cnpm run test
```

### 整理和修复文件
```
cnpm run lint
```

### 自定义配置
点击 [Configuration Reference](https://cli.vuejs.org/config/).
